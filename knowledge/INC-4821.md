# Incident Report: INC-4821

## Author: jagadeesan.vg@cognizant.com - 2276259
## Date: 2026-03-15
## Severity: Medium
## Database: salesdb
## Source: Azure SQL Managed Instance
## Target: SQL Server 2025 VM (10.10.0.5)

## Issue
Restore failed with error: "There is insufficient free space on disk volume. The database needs at least 10240 MB of free space."

## Root Cause
Target drive had only 3GB free space. The salesdb backup required 10GB for data files.

## Resolution
1. Cleaned old backup files on target drive (freed 15GB)
2. Reran restore successfully

## Lesson Learned
Always verify free disk space >= 1.5x backup size BEFORE starting restore.

## Prevention
Added mandatory disk space pre-check to SOP-RESTORE-001 Step 1.
